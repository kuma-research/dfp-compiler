Member Enumeration Documentation

enum DFP::Node::Type

An Enum for Node object type.

Types can be divided into 2 categories: Input/Output and Binary

Enumerator
In 

This Node is an input node, has empty value list.

Out 

This Node is an output node, has 1 value in the value list

Add 

This Node will add up two input values

Sub 

This Node will substract two input values as [0] - [1]

Div 

This Node will multiply two input values

Mult 

This Node will divide two input values as [0] / [1]

Null 

This Node can't be recognized
